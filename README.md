# Tela de cadastro

## Objetivos
- Criar uma tela para cadastro de animais.
- facilitar a agenda de petshops e clinicas veterinárias.

## Funcionalidades
1. Agendar o nome idade e raça do animal.
2. Criar uma lista que pode ser atualizada.
